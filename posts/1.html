1.Hexo介绍

1.1 什么是Hexo

Hexo 是⼀款基于 Node.js 开发的博客框架系统，支持使用 Markdown 语法编写文章，让用户能够专注于内容创作本身。通过 Hexo 的引擎，可以将创作好的 Markdown 文章转换为 HTML 静态页面，这⼀过程通常只需要几秒钟。（Hexo 就是⼀个静态网站生成工具。）

Hexo 提供了丰富的主题和插件，能够灵活扩展网站的功能与样式，满足用户的个性化需求。Hexo 官方站点：https://hexo.io/zh-cn/

1.2 使用Hexo

通常我们会选择在 Windows 下安装 Hexo，本地通过 Typora 加载存放文章的源目录，而后进行文章的创建，文章创作完成后，通过 Hexo 命令一键生成静态网页，并将生成的网页通过插件推送到 GitHub 或服务器上完成文章的发布。
